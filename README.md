 # Rock-Paper-Scissors Game 🎮
 ** Live Demo:** https://mohdali644.github.io/Rps--game/

An interactive **Rock-Paper-Scissors** game built with **HTML, CSS, and JavaScript**.  
This project demonstrates core front-end skills including DOM manipulation, event handling, and responsive UI design packaged in a clean, minimalist interface with a live scoreboard.

---

## ✨ Features
- 🎯 **Interactive Gameplay** – Play Rock, Paper, or Scissors against the computer.
- 📊 **Scoreboard** – Tracks points for *Player* vs *Computer* in real time.
- ⚡ **Dynamic Results** – Displays round outcomes (Win, Lose, Draw).
- 📱 **Responsive Layout** – Optimized for both desktop and mobile screens.
- 🎨 **Minimalist UI** – Simple design with clear icons and intuitive flow.

---

## 🛠️ Tech Stack
- **HTML5** – Semantic structure and layout  
- **CSS3** – Styling, responsiveness, and visual polish  
- **JavaScript (ES6)** – Game logic, interactivity, and DOM updates
